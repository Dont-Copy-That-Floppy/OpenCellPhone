## Arcane AT Commands

| Command                                   | Description                |
|-------------------------------------------|----------------------------|
| `ATD*99#`                                 | Dial Access Point          |
| `AT+CGDCONT=1,"IP","access.point.name"`   | Defines PDP context        |
| `AT+VRX`                                  | Record in-call audio       |
| `AT+COPS=?`                               | List available networks     |
